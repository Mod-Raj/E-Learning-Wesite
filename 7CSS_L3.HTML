  3.CSS Font Size:
  CSS Font size property is used to change the size of the font.
  This are the possible values that can be used to set the font size:
  
  

    

        Font Size Value
        Description
    

    

        XX-small
        Used to display the extremely small text size.
    

    

        X-small
        Used to display the extra small text size.
    

    

        small
        Used to display small text size.
    

    
    

        medium
        Used to display medium text size.
    

    

        large
        Used to display large text size.

  4. CSS Font Styl :
  CSS Font style property defines what type of font you want to display. It may be italic, oblique, or normal.
  

  5.CSS Font Variant
  CSS font variant property specifies how to set font variant of an element. It may be normal and small-caps.
 

 6. CSS Font Weight
 CSS font weight property defines the weight of the font and specify that how bold a font is. The possible values
 of font weight may be normal, bold, bolder, lighter or number (100, 200..... upto 900).
